Bot Setup Instructions
  1. Download Telegram and create an account
  2. Search for @BotFather
  3. Type /newbot into the chat box
  4. Follow the instructions conveyed
  5. Copy the api key provided by BotFather
  6. Paste the API Key and submit it in the Subscriptions page
  7. Send a message saying "/iamroot", this will link you with this chatbot
