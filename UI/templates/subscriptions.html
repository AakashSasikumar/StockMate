What are subscriptions?
Currently there is no support for automated trading, so the next best thing is a personal chatbot to tell you when to buy and sell stock. From the Create Agents page, agents can be made and subscribed to. Once subscribed, it will send messages regarding what orders are to be placed by you. To set this up, you need to download the Telegram chatting application and follow these instructions to set up a bot and provide your api key below.
